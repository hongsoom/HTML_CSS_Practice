## 링크 Anchor

```HTML
현 위치에서 다른 위치로 이동할 때 사용하는 태그 <a>
하지만 꼭 필요한 속성이 있는데 `href` 속성을 꼭 적어줘야 한다. <a href="주소">
href에는 ① URL ② 상대경로 ③ 메일 쓰기 ④ 전화걸기 등 들어갈 수 있다.
③ 메일 쓰기 <a href="mailto:메일주소">
④ 전화걸기 <a href="tel:전화번호">

➕ 또 다른 속성 `target`
<a target="_blank"> : 새로운 tab을 열어서 이동
```
