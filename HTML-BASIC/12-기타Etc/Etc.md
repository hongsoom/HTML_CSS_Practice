## 기타 Etc

### ① abbr
마우스를 올리면 약자가 어떤건지 나타나는 태그
```HTML
<abbr title="abbr의 풀네임">약어</abbr>
```

### ② address
물리적 주소, URL, email주소, 전화번호, sns주소 등등

```HTML
<address>연락처</address>

<address>
	<h1>수민</h1>
	<a href="https://velog.io/@hongsoom">https://velog.io/@hongsoom</a>
</address>
```

### ③ pre, code
pre : 내가 친 그대로 나오는 태그 (줄바꿈, 띄어쓰기)
code : pre태그 안에다가 대부분 넣어짐
```HTML
<pre> 
	홍수민 
            홍수민
</pre>
<pre> 
	<code>
    	홍수민
         홍수민
    </code>
</pre>
```
